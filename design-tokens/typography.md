# Typography Tokens — Limitless Modus

Last updated: 2026-03-05

## Font Family

| Token | Value | Usage |
|-------|-------|-------|
| `--lm-font-family` | `'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif` | All UI text |

Load **Inter** from Google Fonts (weights 400, 500, 600, 700).

## Type Scale

| Token | Size | Line Height | Weight | Usage |
|-------|------|-------------|--------|-------|
| `--lm-font-size-xs` | `11px` | `16px` | 400 | Badges, fine print, footer |
| `--lm-font-size-sm` | `12px` | `18px` | 400 | Captions, labels, sidebar items |
| `--lm-font-size-base` | `14px` | `22px` | 400 | Body text, form fields, buttons |
| `--lm-font-size-md` | `15px` | `24px` | 400 | Slightly larger body (question text) |
| `--lm-font-size-lg` | `16px` | `26px` | 500 | Section titles, card headings |
| `--lm-font-size-xl` | `20px` | `30px` | 600 | Page titles |
| `--lm-font-size-2xl` | `24px` | `32px` | 700 | Dashboard headings |

## Font Weights

| Token | Value | Usage |
|-------|-------|-------|
| `--lm-font-weight-normal` | `400` | Body text |
| `--lm-font-weight-medium` | `500` | Labels, semi-emphasis |
| `--lm-font-weight-semibold` | `600` | Buttons, headings |
| `--lm-font-weight-bold` | `700` | Strong headings |
