# Spacing, Radii & Shadows — Limitless Modus

Last updated: 2026-03-05

## Spacing Scale

Based on a 4px grid. Use these tokens for all padding, margin, and gap values.

| Token | Value | Usage |
|-------|-------|-------|
| `--lm-space-2xs` | `4px` | Tight gaps (icon margin, inline spacing) |
| `--lm-space-xs` | `8px` | Compact padding, small gaps |
| `--lm-space-sm` | `12px` | Input padding, sidebar item spacing |
| `--lm-space-md` | `16px` | Standard content padding, form gaps |
| `--lm-space-lg` | `20px` | Section padding, responsive mobile |
| `--lm-space-xl` | `24px` | Card padding, layout gutters |
| `--lm-space-2xl` | `32px` | Large section breaks, wide layout padding |
| `--lm-space-3xl` | `40px` | Major layout padding (desktop content area) |
| `--lm-space-4xl` | `48px` | Hero spacing, large separators |

## Layout Constants

| Token | Value | Usage |
|-------|-------|-------|
| `--lm-layout-max-width` | `1440px` | Maximum content width on wide displays |
| `--lm-layout-topbar-height` | `48px` | Top bar / header height |
| `--lm-layout-sidebar-width` | `260px` | Wizard sidebar width |
| `--lm-layout-banner-height` | `40px` | Context banner height |

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `--lm-radius-sm` | `4px` | Buttons, small elements |
| `--lm-radius-md` | `6px` | Inputs, checkboxes |
| `--lm-radius-lg` | `8px` | Cards, login box |
| `--lm-radius-xl` | `12px` | Large panels, dashboard cards |

## Box Shadows

| Token | Value | Usage |
|-------|-------|-------|
| `--lm-shadow-sm` | `0 1px 4px rgba(0, 0, 0, 0.06)` | Subtle lift (inputs, small cards) |
| `--lm-shadow-md` | `0 2px 12px rgba(0, 0, 0, 0.08)` | Standard card elevation |
| `--lm-shadow-lg` | `0 2px 16px rgba(0, 0, 0, 0.06)` | Login card, modals |
| `--lm-shadow-topbar` | `0 2px 8px rgba(0, 0, 0, 0.2)` | Top bar shadow |
