# Colour Tokens — Limitless Modus

Last updated: 2026-03-05

## Brand Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `--lm-color-primary` | `#1a237e` | Brand primary — top bar, headings, primary actions |
| `--lm-color-primary-hover` | `#0d47a1` | Primary hover state |
| `--lm-color-secondary` | `#1565c0` | Secondary blue — links, active states, secondary buttons |
| `--lm-color-secondary-hover` | `#1976d2` | Secondary hover |
